# 🤖 Gemini AI Chatbot (Python + Tkinter)

A simple desktop chatbot application built using **Python**, **Tkinter**, and the **Google Gemini API**.

The application provides a clean graphical interface where users can chat with Google's Gemini AI in real time.

## ⚙ Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/gemini-ai-chatbot-python.git
```

### 2. Navigate to the project

```bash
cd gemini-ai-chatbot-python
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

or

```bash
pip install google-genai
```

---

## 🔑 Configure API Key

Open

```python
geminiChatBot.py
```

Replace

```python
API_KEY = "YOUR_API_KEY"
```

with your own Gemini API key.

> **Important:** Never upload your real API key to GitHub. Store it in an environment variable or a `.env` file.

---

## ▶ Run the Application

```bash
python geminiChatBot.py
```
